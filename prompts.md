# Student Profile Card - Design Prompts

This document tracks the prompts used to generate and iterate on the Student Profile Card project.

## Prompt 1: Initial Creation & Structure
> 1. Prompt AI to generate a Student Profile Card with:
>    - Name
>    - Stream
>    - Short bio
>    - Profile image placeholder
> 2. Save the file as `index.html` and open it in a browser.
> 3. Identify:
>    - `<div>` structure
>    - `<head>` section
>    - CSS styling section
> 4. Improve the design:
>    - Add rounded corners
>    - Add shadow
>    - Change font
>    - Add gradient background
